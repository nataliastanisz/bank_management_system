# bank_management_system

simple bank app made with python. has two versions: cli for the terminal and a web ui for the browser.

### what it does:
*   create account with pin
*   deposits & withdrawals (pin validated)
*   check balance
*   data saved to a json file (database)

### tech:
*   python 3
*   streamlit (for the web version)
*   json

### how to run:
first: `pip install streamlit`

web version:
`streamlit run streamlit_app.py`

terminal version:
`python3 app.py`
